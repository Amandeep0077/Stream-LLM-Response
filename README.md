# 🚀 Stream LLM Responses with LangChain & Groq

A lightweight Python notebook demonstrating how to stream Large Language Model (LLM) responses in real-time using **LangChain** and **Groq API**.

---

## 📌 Features

* ⚡ **Real-Time Streaming:** Print tokens as they are generated for a smoother user experience.
* 🤖 **Groq Integration:** Powered by `langchain-groq` for ultra-fast model inference.
* 🔑 **Environment Management:** Easily load API keys securely with `python-dotenv`.

---

Before running the notebook, ensure you have Python installed along with the required libraries:
